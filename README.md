# RepoSense AI

RepoSense AI is a full-stack AI-powered web application that allows users to understand any GitHub repository using natural language queries.

## Features
- **Git Repository Cloning:** Automatically loads and parses code from any public GitHub repository.
- **Smart Code Chunking & Embeddings:** Processes source files using text embeddings for semantic search via FAISS.
- **AI Chat Interface:** Ask questions about the codebase and get context-aware answers generated by GPT-4o-mini.
- **Architectural Summary:** Automatically generates a high-level summary of the repository structure and technologies.
- **Modern UI:** Built with React, Tailwind CSS, Framer Motion, and Lucide React for a premium look and feel.

## 🚀 How to Run

### Prerequisites
- Node.js (v18+)
- Python (3.9+)
- OpenAI API Key

### 1. Setup Backend
1. Open a terminal and navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Create a virtual environment and activate it (optional but recommended):
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # Mac/Linux
   source venv/bin/activate
   ```
3. Install Python dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Set your OpenAI API Key as an environment variable:
   ```bash
   # Windows
   set OPENAI_API_KEY=your_api_key_here
   # Mac/Linux
   export OPENAI_API_KEY=your_api_key_here
   ```
   *(Alternatively, edit `backend/utils.py` and replace `YOUR_API_KEY` with your key)*

5. Start the FastAPI server:
   ```bash
   uvicorn main:app --reload --port 8000
   ```
   The backend will be running at `http://localhost:8000`.

### 2. Setup Frontend
1. Open a **new** terminal and navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install Node dependencies:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```
4. Open the provided `localhost` URL in your browser (usually `http://localhost:5173`).

## Usage Guide
1. Obtain the URL of a public GitHub repository (e.g., `https://github.com/facebook/react`).
2. Paste the URL into the search bar at the top of the RepoSense AI app.
3. Click **Load Repo**. The backend will clone the repository, process the files, generate embeddings, and build the search index.
4. Once loaded, the left sidebar will display the list of parsed source files.
5. The UI will automatically navigate to the **Repository Summary**, providing an architectural overview.
6. Switch to the **Chat Interface** from the sidebar to ask specific questions like *"How does the routing work in this app?"* or *"Where is the database connection instantiated?"*
